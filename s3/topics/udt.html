Introduction Transferring files between server is no big deal with nowadays network equipments. You use rsync, scp or even http to get a file from A to B. Of course, you rely on the TCP stack so you have a decent reliability in the transport. But TCP has its drawback, especially when it needs to go through a lot of equipments.
