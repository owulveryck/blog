Installation

From sources:

cd $GOPATH/src/github.com/ethereum
git clone https://github.com/ethereum/go-ethereum.git

installing the solidity compiler

sudo add-apt-repository ppa:ethereum/ethereum
sudo apt-get update
sudo apt-get install solc
which solc
